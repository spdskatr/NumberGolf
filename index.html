How to play

Try to make the number shown on the screen using only the characters 1, +, *, ( and ). You are allowed to "concatenate" 2 numbers together, like 11 and 111, but addition and multiplication must be written out explicitly. Some examples of valid and invalid equations:

  • 1*(111+1) (valid)
  • (1+11)(1+11) (invalid because no explicit * between brackets)
  • 11*(1+(1*11))+(1+1) (valid)
  • (1+1)*((1+1+1)*11 (invalid because parentheses are not balanced)
